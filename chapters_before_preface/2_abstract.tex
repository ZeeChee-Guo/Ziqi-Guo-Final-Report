\prefacesection{Abstract}

Time-series anomaly detection is difficult when the distribution of data changes over time. Unusual patterns may indicate true anomalies, but they may also reflect concept drift, where the normal data distribution evolves. Treating drift as anomalies can generate many false alarms. To remain effective under changing conditions, anomaly detection methods often adapt their model of normal behavior using new observations. However, adapting too quickly may cause true anomalies to be incorporated into the learned normal pattern, making them harder to detect.

Andri is a time-series anomaly detection algorithm that compares subsequences with normal patterns that may change as the data evolves. The method also allows different thresholds to be applied to different normal patterns under concept drift. Therefore, this method tends to perform better when both time series anomalies and concept drift occur simultaneously \citep{park2025adaptive}. Building upon the original AnDri algorithm, this project develops an interactive demonstration system for anomaly detection under concept drift. The system provides visualization, user interaction, feedback collection, and comparative evaluation capabilities, allowing users to explore anomaly detection and concept drift in a practical environment. 

The implemented system consists of a browser-based interface and a backend based on Python Flask. With this system, baseline methods including NormA, SAND, and DAMP were used to compare the time series anomaly effects with the Andri algorithm on different datasets and conduct visual exploration evaluations. The system demonstrated how interactive feedback and threshold adjustment based on user feedback can help distinguish true anomalies from changes in normal behavior.
